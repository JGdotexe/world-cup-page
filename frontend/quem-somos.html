Sobre o projeto
O problema
Acompanhar e palpitar em um só lugar

Durante grandes campeonatos, torcedores não têm uma plataforma gratuita e unificada para acompanhar os jogos em tempo real e registrar seus palpites para compartilhar com amigos.

A solução
Bolão da Copa 2026

Uma aplicação web que consome a API pública da ESPN para exibir dados em tempo real — placares, tabelas de grupos e estatísticas — e permite que cada usuário registre e compartilhe seus palpites via link único.

Tecnologias utilizadas
HTML5 CSS3 Bootstrap 5 JavaScript (ES6+) PHP 8.2 MySQL 8 ESPN API Docker
Arquitetura
Backend · PHP + MySQL
API de armazenamento

O servidor PHP gerencia exclusivamente autenticação de usuários (sessões) e persistência dos palpites no banco MySQL. Não se comunica com a ESPN — isso mantém o backend leve e compatível com hospedagem gratuita.

Frontend · JavaScript
Motor de exibição

O JavaScript rodando no navegador do usuário faz as chamadas à API pública da ESPN, recebe os dados dos jogos, cruza com os palpites salvos no nosso backend e exibe tudo na tela dinamicamente.
